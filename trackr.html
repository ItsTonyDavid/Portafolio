Trackr.

Website for managing your tasks into categories and subtasks in order to be more productive and complete all of your pending activities. This website also have the capacity to share dashboards with teams, to work along with teammates.


Full-stack developer.

Fully developed the front-end of the website and connecting to the database in order to do CRUD functionalities.
Participate in the development of the backend of the website along with a teammate, as well as in the hosting and creating the Data Base. the project on Heroku and the database on MongoDB.


Tools and Technology

Front-end: HTML, CSS with Bulma framework and Javascript with jQuery.
Back-end: NodeJS, Heroku for the hosting and MongoDB for database in the cloud.
